FAQ / Règles

Éligibilité, validation badge/code, données personnelles.

Qui peut utiliser le Pass Boréales ?

Accès réservé aux collaborateurs disposant d'un badge actif ou d'un code communiqué par l'équipe CSE.

Badge ou code : que présenter ?

Chaque fiche précise « Badge », « Code » ou « Badge ou code ». En cas de doute, présenter les deux informations.

Validité des offres

Les offres sont susceptibles d'évoluer. La date « Dernière mise à jour » est affichée sur la fiche.

Refus d'un partenaire

Utilisez « Signaler un problème » dans la fiche. Le support contacte le partenaire sous 24h.

Données personnelles

Aucune donnée personnelle n'est transmise aux partenaires. Les codes sont génériques ou temporaires.
